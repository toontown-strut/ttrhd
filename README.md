# ttrhd
Toontown Rewritten HD Content Pack by NodeMixaholic and the Toontown Strut Team.

## Instructions

1. download latest release (.mf file)
2. make a new folder in your toontown rewritten directory called "resources" (no quotes)
3. put it in
